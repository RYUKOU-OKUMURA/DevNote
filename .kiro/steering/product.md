# Product Overview

DevNote は、GitHubリポジトリをAIが学習し、ユーザーがNotebook LM風のワークスペースでコードやドキュメントについて対話的に質問できるRAG（検索拡張生成）アプリケーション。

## Core Capabilities

- **リポジトリ学習**: GitHubリポジトリを自動インデックス化し、Gemini APIのFile Search機能でRAG検索を実現
- **統合ワークスペース**: ファイルツリー、チャット、メモパッドの3要素を1画面に統合したシームレスな調査体験
- **根拠の明示**: AIの回答には引用元ファイルを必ず明記し、Citation機能で信頼性を担保
- **コンテキスト分離**: 1リポジトリ = 1ノート（ワークスペース）として、チャット履歴やメモを完全に分離管理

## Target Use Cases

- **コードベース理解**: 新規参加プロジェクトやOSSリポジトリの迅速な理解
- **アーキテクチャ調査**: 特定ファイルや関数の依存関係、設計意図の探索
- **ドキュメント検索**: READMEやコメントから実装の背景知識を抽出
- **知見の蓄積**: 調査中に得た気づきをメモパッドに記録し、ノート単位で永続化

## Value Proposition

- **ワークスペース型UI**: 従来のチャットボットではなく、Notebook LMのような専用環境でリポジトリに没入
- **引用ベースの信頼性**: Gemini File Searchによる正確な引用で、AIの「幻覚」を最小化
- **GitHub統合**: OAuth認証により、プライベートリポジトリも含めてシームレスにアクセス
